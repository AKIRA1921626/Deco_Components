# Deco_Components
装飾セットの各種機能を小分けにしたスクリプトたち  
小分けにするのと同時に、一部機能を変更したり、機能を追加したり、改良したりしている。  

## 装飾セット.anm2
https://github.com/AKIRA1921626/text-decoration-set  
このスクリプトの各種機能をわけています。

### deco_Color_Tex.anm2
これはテキストの文字の色を変更するスクリプトです。  
普通の文字の色の変更とは違い、個別オブジェクトが有効の時、3つの範囲にそれぞれ違う色を指定することが可能です。  
例:1から3文字までは文字色1,4から6文字までは文字色2,7から9文字までは文字色3  

### deco_Grad_Tex.anm2
これはテキストにグラデーションをかけられるスクリプトです。  
普通のグラデーションとの違いは、テキストを個別オブジェクトにした場合、2か所にそれぞれ別のグラデーションをかけられることです。  
例:1から3文字まではグラーション1,5から6文字まではグラデーション2  

### deco_Effect_Glow.anm2
これはテキストの後方にグローを発生させられるスクリプトです。  
普通のグローは文字の端の方を巻き込んで一緒に光っていましたが、これは文字はそのままに後ろだけが光るようになっています。  

### Integrate_text.anm2
これは様々なスクリプトに存在する「テキスト1枚化」のチェックボックスを、このスクリプトで行えるように分離したものです。  
チェックボックスで機能するのではなく、スクリプトで機能する形に変更しただけなので、AKIRA_aviutl2_helper.luaが同じフォルダに必要です。  

### Individual_object_join.lua
これは各種スクリプトに存在する「テキスト1枚化」の機能を有効にするための物です。  
「テキスト1枚化」の機能を使いたい場合、このファイルをそれらのスクリプトと同じフォルダに入れてください。 
このファイルは1つだけあれば問題ありません。  

### AKIRA_aviutl2_helper.lua
これは各種スクリプトを機能させるための物です。  
ほとんどのスクリプトはこのファイルが無ければ動きません。  
このファイルは各種スクリプトと同じフォルダに入れてください。  
このファイルは1つだけあれば問題ありません。  
また、これはluaで書かれており、分けて書いた方が書きやすいという作成者の都合で作成したものなので、中身を適切な位置にコピペすれば、どのスクリプトも単体で機能するようになります。  

#### 二次配布可、改変可、クレジット不要、各種.luaに関しても同様となります。
